## Project Overview
This project aims to investigate the philosophy, ecosystem, and licensing of open-source software, using Linux as the main example. Additionally, there are helpful shell scripting exercises to show off your command-line skills with Linux.Cript, take these actions:

## Scripts Included

### 1. System Identity Report
Displays system info like kernel, user, uptime.

### 2. FOSS Package Inspector
Checks if a package is installed and shows details.

### 3. Disk and Permission Auditor
Analyzes directories, permissions, and storage usage.

### 4. Log File Analyzer
Counts occurrences of keywords in log files.

### 5. Open Source Manifesto Generator
Generates a personalized open-source statement.
